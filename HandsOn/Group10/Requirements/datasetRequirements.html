Dataset Requirements

R1: Smart City Domain

Cumple directamente con el requisito ya que nuestro dataset afecta al dominio de uno de los requisitos para las smart city, en este caso, "Transporte público y movilidad urbana sostenible".

R2: CSV File Availability

Cumple el requisito ya que el dataset se encuentra disponible en formato CSV.

R3: Open License

Nuestro dataset sí es de licencia abierta, concretamente se puede compartir, transformar o modificar

R4: Link with Real-World Entities

Cumple el requisito ya que los parques de bici se pueden localizar fácilmente con diversas webs, como por ejemplo la más famosa a día de hoy, Google Maps.

R5: Documentation (Optional)

Nuestro dataset no cuenta con documentación adicional por lo que no cumple el requisito

R6: Multiple Data Sources (Optional)

Nuestro dataset no cumple con este requisito ya que solo cuenta con una fuente fiable
