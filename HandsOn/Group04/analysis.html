DATA DESCRIPTION:
This dataset contains eleven different columns: the name of the district, the OND code, the region where the district is located, and seven columns with data from 2001 to 2008.
Both the district and the OND code are unique for each row. After purging and cleaning the dataset, we successfully refined it down to a total of 316 records. We can find 8 different regions, along with the corresponding number of rows belonging to each region:E Midlands 40, Eastern 48, London 33, North East 23, North West 43, S East 67, S West 45, W Midlands34, Yorkshire/Humber 20. The numerical data in the last seven columns are integer values, representing the number of abandoned cars.

For each individual year the values will be in the following characteristics (blank spaces represent null values):
 2001: range between 0 - 6800
 2002: range between 0 - 8200
 2003: range between 0 - 6000
 2004: range between 0 - 4400
 2005: range between 0 - 3200
 2006: range between 0 - 6000
 2007: range between 0 - 5300
 2008: range between 0 - 1800
