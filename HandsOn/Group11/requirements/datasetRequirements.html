Dataset Requirements

  • R1. Data are in one of the selected smart city domains.

    The data was sourced from Madrid Open Data and represents smart city data for air quality.

  • R2. Data are available as a CSV file.

    The data is available for download from the site, marked as horarios.csv and informacion_estaciones_red_calidad_aire.csv under the csv folder.

  • R3. Data have an open license so they can be published.

    The data has an open license that allows it to be published: Condiciones generales

  • R4. Data can be easily linked with generic real-world entities (e.g., locations).

    The data can be easily linked with each of the Control Stations it was sourced from, also available in this dataset this dataset.

  • R5. Documentation exists for the data (optional).

    There is ample documentation about the data, freely available at this document.

  • R6. Data can come from multiple data sources (optional).

    As mentioned previously, we will use the dataset for the Control Stations for their geo-location data in order to display them on a map.

    In addition to that, the site has other datasets that would integrate naturally into this project, like parks, road traffic, etc.
