Dataset Analysis - SAMUR Activations

1. Data Analysis

Column Description
Año The year when the SAMUR activation occurred.
Mes The month when the SAMUR activation occurred.
Hora Solicitud The time when the SAMUR activation was requested.
Hora Intervención The time when the SAMUR intervention started.
Código The emergency type code or description of the incident attended (e.g., heart attack, convulsions).
Distrito The district in Madrid where the emergency occurred.
Hospital The hospital to which the patient was taken (if applicable). This column may be empty if no hospital intervention was required.

Ranges and Quantities of Interest

  • Años: All data was gathered during 2023, so all values are this exact year.
  • Meses: Activations recorded for each month in 2023 (January to December).
  • Hora Solicitud: Time of request. Values are in format HH:MM:ss in 24H format.
  • Hora Intervención: Time of intervention, giving insights into the speed of response. Values are in format HH:MM:ss in 24H format.
  • Código: Types of emergencies handled by SAMUR (e.g., Infarto, Convulsión). There are 90 emergency codes used during this year.
  • Distrito: Covers 25 districts of Madrid where emergencies were recorded.
  • Hospital: Indicates whether the emergency led to hospitalization and which of the 19 recorded hospitals received the patient.

2. Licensing of the Data Source

The dataset is published by the Madrid City Council, and the rightsholder is the same.

The dataset is available under an open license, allowing for both commercial and non-commercial reuse.

For the generated dataset, we will use the same open license to ensure compliance and consistency.
